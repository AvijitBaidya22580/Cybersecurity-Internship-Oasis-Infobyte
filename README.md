## 🧠 About This Repository

<p align="justify">
This repository documents my <b>Cybersecurity Internship at Oasis Infobyte</b>, where I performed hands-on security testing, vulnerability assessment, and network traffic analysis in a controlled lab environment.

The projects simulate <b>real-world cyber attack and defense scenarios</b>, covering:

</p>

* 🔍 Vulnerability Scanning
* 💉 SQL Injection Exploitation
* 🌐 Network Traffic Analysis
* 📊 Security Reporting

<p align="justify">
Each task is structured as a <b>mini case study</b> with step-by-step execution, screenshots, findings, and security insights.
</p>

## 🚀 Completed Tasks

### 💉 Task 3 – SQL Injection (Core Skill)

<p align="justify">
Performed practical exploitation of SQL Injection vulnerability using DVWA. Demonstrated authentication bypass, database extraction, and backend query manipulation.
</p>

🔗 **View Project:** `Task-3-SQL-Injection/`

---

### 🔎 Task 7 – Vulnerability Scanning with Nikto

<p align="justify">
Conducted automated web server vulnerability scanning using Nikto. Identified outdated software, missing security headers, and potential misconfigurations.
</p>

🔗 **View Project:** `Task-7-Nikto/`

---

### 🦈 Task 8 – Network Traffic Analysis with Wireshark

<p align="justify">
Captured and analyzed real-time network traffic including DNS queries, TCP handshakes, TLS encryption, and QUIC protocol communication.
</p>

🔗 **View Project:** `Task-8-Wireshark/`

---

### 📊 Task 10 – Security Report (Final Deliverable)

<p align="justify">
Compiled a professional cybersecurity report summarizing vulnerabilities, attack vectors, and mitigation strategies based on all performed tasks.
</p>

🔗 **View Project:** `Task-10-Security-Report/`
